# 快速开始

多摄像头与视频演示模式说明。

## 已支持能力

- **多本地摄像头**：自动检测 USB/内置摄像头，支持多设备切换并记住设备 ID。
- **网络摄像头**：MJPEG/HLS 流，健康检查与自动重连。
- **视频文件输入**：本地视频作为输入源，自动循环，适合演示。
- **摄像头管理**：后端持久化、前端管理、状态监控。

## 3 分钟演示

1. **准备**：准备包含清晰车牌的 MP4（约 30 秒–2 分钟）。
2. **启动**：`.\start-dev.ps1`（或分别启动 ai-service、backend、frontend）。
3. **添加视频源**：打开 http://localhost:5173 → 实时监控 → 摄像头列表「+」→ 选「视频文件」→ 填名称、选文件 → 添加。
4. **识别**：在列表中选择该视频摄像头 → 点击「开启识别」。
5. **查看**：界面实时显示结果；记录管理看历史；数据仪表盘看统计。

## 接入真实摄像头

- **USB**：连接设备 → 摄像头列表点「刷新」→ 选择后点「开启识别」。
- **网络流**：列表「+」→ 选「网络流」→ 填名称与流地址。常见格式：MJPEG `http://IP:端口/video.mjpg`，HLS `http://IP:端口/stream.m3u8`。

## 性能与设置

默认配置（置信度 70%、扫描间隔 2s、同车防重复 30s、漏检补拍开启）适合多数场景。可在右上角设置中调整：

- **扫描间隔**：快速 1000–2000ms，标准 2000–3000ms，省电 3000–5000ms。
- **置信度**：高精度 0.8–0.9，标准 0.7，演示可降至 0.6。

## 常见问题

- **视频无法播放**：使用 MP4/WebM 等浏览器支持格式。
- **网络摄像头连不上**：核对流地址、浏览器直接打开 URL 测试、检查网络与认证。
- **识别效果差**：保证车牌清晰、调整角度与光照、可适当降低置信度。
- **多摄像头**：当前一次仅识别一个源，可在列表中快速切换。
