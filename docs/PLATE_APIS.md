# 车牌相关线上接口接入参考

本文档汇总可接入的**车牌/车辆信息**类第三方 API，用于在识别到车牌后做归属地、车辆属性等增强（可选）。

> **注意**：以下接口均为第三方服务，需自行注册、付费或遵守免费额度；数据与隐私政策以各平台为准。本系统不内置任何密钥，接入时请在服务端配置并做好鉴权与限流。

---

## 国内接口

### 1. 车牌归属地（免费）

| 说明 | 接口示例 | 返回内容 |
|------|----------|----------|
| **夏柔 API** | `GET https://v.api.aa1.cn/api/car-number-fl/index.php?car_number=京A12345` | 省份、城市、组织单位、车牌类型（民用/军用/使馆等） |

- 适合：仅需“车牌归属地/类型”时使用。
- 集成方式：后端在保存记录前后可选调用，将返回的省份/城市写入 `region_code` 或扩展字段。

### 2. 聚合数据 · 车牌归属地

- **文档**：[车牌归属地查询 - 聚合数据](https://juhe.cn/docs/api/id/728)
- **请求**：GET/POST，参数 `key`（API Key）、`number`（车牌号）。
- **返回**：省份、城市、车牌类型等。
- **计费**：有免费额度，超出按次计费。

### 3. 聚合数据 · 车五项（车牌查车辆信息）

- **文档**：[车五项 - 聚合数据](https://juhe.cn/docs/api/id/812)
- **请求**：`GET http://apis.juhe.cn/vehicle_five/query?key=KEY&carNumber=车牌号`
- **返回**：车辆品牌、VIN、车型、发动机号、运营性质、初次登记日期等（**不含车主个人信息**）。
- **计费**：按次计费，约 0.1 元/次起。
- **适合**：需要“识别到车牌后展示车辆基本信息”的场景。

### 4. 挖数据 · 车牌号查车辆信息（精准版）

- **说明**：返回初始登记日期、上险日期、保险到期、车架号、品牌等（不含车主信息）。
- **计费**：约 4.5–6 元/次，量大有优惠；数据更新周期较长（如每月）。

### 5. 阿里云 / 腾讯云 车辆相关 API

- 阿里云云市场、腾讯云等有车辆 VIN 码查询、车辆信息等 API，可按“车牌→VIN→车辆信息”或平台提供的车牌直达接口接入，具体以云市场文档为准。

## 国际接口（多用于海外车牌）

| 服务 | 说明 |
|------|------|
| **CarsXE** | 支持多国车牌，返回 VIN、品牌、型号、年份等，需 API Key。 |
| **NHTSA vPIC** | 美国官方 VIN 解码，免费，需先由车牌反查 VIN（通常需配合其他服务）。 |
| **Vehicle Databases 等** | 美/英等地区车牌解码，按次计费。 |

本系统当前主要面向**中国车牌**，若需支持海外车牌，可在后端增加“国家/地区”参数并路由到对应接口。

## 在本项目中的可选集成方式

1. **仅展示归属地**  
   识别到合法车牌并保存后，后端异步调用“车牌归属地”接口，将省份/城市写入 `plate_records` 的 `region_code` 或扩展字段，前端在列表中展示。

2. **车辆信息增强**  
   需要品牌/型号等信息时，可调用“车五项”或同类接口，将结果缓存到数据库或 Redis（按车牌号 + 一定过期时间），前端在详情页展示。

3. **实现建议**  
   - 在 `backend` 中新增 `services/plateLookupService.ts`（或类似），封装上述 HTTP 调用。  
   - 使用环境变量配置 API Key 和开关（如 `ENABLE_PLATE_LOOKUP=1`、`JUHE_API_KEY=xxx`）。  
   - 在 `PlateController.savePlate` 或 `savePlateRecord` 之后，根据配置决定是否调用 lookup 并写回扩展信息。  
   - 做好限流、超时与错误处理，避免第三方不可用影响主流程。

4. **隐私与合规**  
   仅使用“车辆属性/归属地”类接口，不接入、不存储车主姓名、电话、身份证等个人信息；若业务涉及敏感数据，需另行评估合规要求。

## 小结

| 需求 | 可选用 |
|------|--------|
| 免费、只要归属地/类型 | 夏柔 车牌归属地 |
| 国内、要车辆品牌/型号等 | 聚合 车五项 或 挖数据 精准版 |
| 海外车牌 | CarsXE、NHTSA、各区域商用 API |

按需选择其一或组合使用，在后台用环境变量和开关控制即可，无需改识别与存储主流程。
