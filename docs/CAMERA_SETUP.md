# 摄像头接入与演示

## 本地摄像头（USB/内置）

- 摄像头列表点「刷新」自动检测设备，选择后即可使用。
- 支持多 USB、前后置切换、实时流与车牌识别。

## 网络摄像头（IP / RTSP）

- 当前版本已内置后端流代理：统一接收 RTSP/HLS/HTTP 流，并由后端转成浏览器可播放的 MJPEG。
- 生产环境请确保后端机器已安装 `ffmpeg`（命令行可执行 `ffmpeg -version`）。
- 添加：列表「+」→ 网络流 → 填名称与流地址。常见格式示例：海康 `http://用户:密码@IP:端口/Streaming/channels/1/picture`；通用 MJPEG `http://IP:端口/video.mjpg`。
- 已提供“智能接入助手 + 连通测试”：可按厂商（海康/大华/宇视/Axis）自动生成常见 RTSP 地址并直接测试可用性。

## 视频文件（演示）

- 列表「+」→ 视频文件 → 填名称、选文件（mp4/webm 等）→ 添加。可多视频切换演示。

## 演示建议

- 推荐先用视频文件保证稳定，再接真实摄像头展示实时效果；可混合配置后快速切换。
- 扫描间隔：实时 1–3s，演示 2–5s，低性能设备 3–5s。置信度：高精度 0.8–0.9，标准 0.7，演示可 0.6。
- 当前一次仅识别一个源，多摄像头通过列表切换。

## 故障排查

- **本地摄像头不可用**：检查浏览器权限、设备连接、是否被其他应用占用，再点「刷新」。
- **网络流连不上**：核对 URL、浏览器直连测试、网络与认证、CORS。
- **识别效果差**：调整角度与光照、扫描间隔、适当降低置信度。

详细步骤与性能说明见 [QUICK_START.md](QUICK_START.md)。
